Buttons in mars are denoted by mars-button.
mars-button can apply gradient, animation, outline, loading, icon.

Overview

mars-button is initialized with a fixed 4px round value by default.

  • is-nor-rounded
  • is-md-rounded
  • is-lg-rounded

Hovered Buttons

mars gives hover the functionality of a button, put is-hovered. Must be placed after mars-button

Overview

Each color is available.Please refer to the detailed color chart here.

Focused Button

The function of focuse uses is-focused, but it is not an event on click.

Overview

The functionality of is-focused applies to border by default.

Pressed Button

pressed uses is-pressed. mars contains is-actived, which has the same functionality as is-pressed.

Overview

is-pressed is subject to a border inset.

Disabled Button

disabled for mars uses is-disabled. This is the same as setting the disable attribute with mars-button applied.

Overview

disabled can be customized as desired.

Loading Button

Set the loading button to is-loading   The animation for loading uses the spinAround name, which is 500ms by default.

Overview

is-loading and use fontawesome by default, you must use fontawesome together. Include it in your project like this Be careful to insert it after jquery.

Gradient Button

The gradient button is applied as is-gradient. By default, the left color is darker than the color on the right. border is not specified. You can also add animation to it using is-anim, so be sure to set it with is-gradient.

Overview

The gradient can only be applied to the color specified in mars.

Anim Button

Anim is a animated feature of gradients. is-anim uses the animation name bgAnim.

Overview

When using is-anim, please be careful about background-size. It is maintained at 400 x 400. Please be careful when changing.

Sized Button

size consists of small, normal, medium, large, and consists of the following classes The unit is specified in rem. Note the font size.

  • is-small
  • is-normal
  • is-medium
  • is-large


0.75rem, 1rem, 1.25rem, 1.5rem respectively.

Overview

Keep it at 16px by default.

State Button

The state, as you know, consists of primary, success, warning, and error. Just add color to the front to distinguish it from other libraries.

  • is-primary
  • is-success
  • is-warning
  • is-error

Overview

Note that state starts with is-color-.

Icon Button

The icon uses is-icon. The icon in mars uses fontawesome. Be sure to use fontawesome if you want to use it.

Overview

You need to use fontawesome, always insert the jquery in front of it.

Outlined Buttons

outline uses is-outlined, by default border is none.

Overview

outlined cannot specify a background color.
